# Guía de Desarrollo - SISFAC

Este conjunto de documentos describe la estructura del código, los módulos principales y los flujos internos del sistema SISFAC desde el punto de vista de desarrollo.

## Índice

- `backend.md`  
  Descripción de la arquitectura del backend (Flask), modelos, blueprints y rutas principales.

- `frontend.md`  
  Descripción de la capa de presentación basada en templates Jinja2, Tailwind CSS y Alpine.js, incluyendo el layout base y componentes interactivos.

- `flujos.md`  
  Flujos funcionales clave explicados desde el código: creación de clientes, gestión de inventario, facturación, historial y talonarios.

Cada archivo está pensado para que un desarrollador pueda entender rápidamente “dónde está cada cosa” y qué papel cumple dentro del sistema.
